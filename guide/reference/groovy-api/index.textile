p. This section describes the "Groovy":http://groovy-lang.org/ API elasticsearch provides. All of elasticsearch APIs are executed using a "GClient":client.html, and are completely asynchronous in nature (either accepts a listener, or return a future).

p. The Groovy API is a wrapper on top of the "Java API":/guide/reference/java-api exposing it in a groovier manner. The execution options for each API follow a similar manner and covered in "the anatomy of a Groovy API":anatomy.html.

h2. Maven Repository

p. elasticsearch is hosted on "Sonatype":http://www.sonatype.org/, with both a "releases repo":http://oss.sonatype.org/content/repositories/releases/ and a "snapshots repo":http://oss.sonatype.org/content/repositories/snapshots.
